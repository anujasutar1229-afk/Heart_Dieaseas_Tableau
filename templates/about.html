About This Project

Understanding heart disease patterns through data analytics

Project Objective

The main goal of this project is to analyze heart disease data and identify risk factors such as age, cholesterol, blood pressure, and lifestyle habits.

Technologies Used

This project uses Tableau for data visualization, Flask for web integration, and Bootstrap for building a responsive and professional interface.

Dataset Insights

The dataset contains various health parameters including age, cholesterol levels, blood pressure, heart rate, and other medical indicators.

Why This Project Matters

Heart disease is one of the leading causes of death worldwide. By analyzing healthcare data through interactive dashboards, this project helps identify patterns and trends that can support better decision-making in healthcare analysis.

The integration of Tableau dashboards into a web application allows users to explore insights dynamically and understand the key factors contributing to heart disease.
